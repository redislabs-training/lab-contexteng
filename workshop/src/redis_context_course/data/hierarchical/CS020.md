# CS020: Big Data Technologies

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: advanced
- **Format**: in_person
- **Instructor**: Carla Gray
- **Semester**: spring 2025
- **Max Enrollment**: 34

## Description

Hadoop, Spark, and distributed data processing. This course covers big data architectures, MapReduce, Spark, and stream processing. Students will process large-scale datasets using distributed computing frameworks.

## Prerequisites

- **CS003**: Prerequisite for Big Data Technologies (Minimum grade: C)

## Learning Objectives

- Understand core concepts in big data technologies
- Implement big data technologies algorithms and techniques
- Apply big data technologies to real-world problems
- Analyze and evaluate big data technologies solutions
- Design and build complete computer science systems
