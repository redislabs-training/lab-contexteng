# CS008: Operating Systems

## Course Information

- **Department**: Computer Science
- **Credits**: 4
- **Difficulty Level**: advanced
- **Format**: in_person
- **Instructor**: Gina Moore
- **Semester**: spring 2025
- **Max Enrollment**: 62

## Description

Process management, memory, file systems, and concurrency. This course explores how operating systems manage hardware resources and provide services to applications. Topics include scheduling, virtual memory, and synchronization.

## Prerequisites

- **CS002**: Prerequisite for Operating Systems (Minimum grade: C)

## Learning Objectives

- Understand core concepts in operating systems
- Implement operating systems algorithms and techniques
- Apply operating systems to real-world problems
- Analyze and evaluate operating systems solutions
- Design and build complete computer science systems
