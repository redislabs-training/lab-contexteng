# CS006: Web Development

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: hybrid
- **Instructor**: Connie Lawrence
- **Semester**: spring 2025
- **Max Enrollment**: 73

## Description

Full-stack web development with modern frameworks. Students will build responsive web applications using HTML, CSS, JavaScript, React, and Node.js. Topics include REST APIs, authentication, and deployment.

## Learning Objectives

- Understand core concepts in web development
- Implement web development algorithms and techniques
- Apply web development to real-world problems
- Analyze and evaluate web development solutions
- Design and build complete computer science systems
