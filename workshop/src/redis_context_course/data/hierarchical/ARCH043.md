# ARCH043: Introduction to Architecture

## Course Information

- **Department**: Architecture
- **Credits**: 3
- **Difficulty Level**: beginner
- **Format**: in_person
- **Instructor**: Sherry Decker
- **Semester**: winter 2025
- **Max Enrollment**: 50

## Description

Fundamentals of architectural design and history. This course introduces students to architectural principles, design thinking, and the history of built environments. Students will develop basic drawing and modeling skills.

## Learning Objectives

- Understand core concepts in introduction to architecture
- Implement introduction to architecture algorithms and techniques
- Apply introduction to architecture to real-world problems
- Analyze and evaluate introduction to architecture solutions
- Design and build complete architecture systems
