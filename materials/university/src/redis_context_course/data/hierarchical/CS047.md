# CS047: Deep Learning and Neural Networks

## Course Information

- **Department**: Computer Science
- **Credits**: 4
- **Difficulty Level**: graduate
- **Format**: online
- **Instructor**: Erica Smith
- **Semester**: summer 2025
- **Max Enrollment**: 58

## Description

Advanced neural network architectures and deep learning techniques. This course dives deep into modern deep learning, covering CNNs, RNNs, transformers, and GANs. Students will implement state-of-the-art models and work on real-world projects. Prerequisites include linear algebra, calculus, and programming experience.

## Prerequisites

- **CS001**: Prerequisite Course 1 (Minimum grade: C)

## Learning Objectives

- Understand core concepts in deep learning and neural networks
- Implement deep learning and neural networks algorithms and techniques
- Apply deep learning and neural networks to real-world problems
- Analyze and evaluate deep learning and neural networks solutions
- Design and build complete computer science systems
