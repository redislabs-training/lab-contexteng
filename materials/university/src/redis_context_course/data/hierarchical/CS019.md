# CS019: Reinforcement Learning

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: advanced
- **Format**: in_person
- **Instructor**: Lisa Jackson
- **Semester**: fall 2025
- **Max Enrollment**: 67

## Description

Agent learning through interaction and reward signals. This course covers MDPs, Q-learning, policy gradients, and deep reinforcement learning. Students will train agents to play games and solve control problems.

## Prerequisites

- **CS010**: Prerequisite for Reinforcement Learning (Minimum grade: C)

## Learning Objectives

- Understand core concepts in reinforcement learning
- Implement reinforcement learning algorithms and techniques
- Apply reinforcement learning to real-world problems
- Analyze and evaluate reinforcement learning solutions
- Design and build complete computer science systems
