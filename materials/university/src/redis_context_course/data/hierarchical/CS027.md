# CS027: Natural Language Processing

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: advanced
- **Format**: in_person
- **Instructor**: Meghan Rhodes
- **Semester**: winter 2025
- **Max Enrollment**: 76

## Description

Processing and understanding human language with machine learning. This course covers text processing, language models, sentiment analysis, and modern NLP techniques. Students will work with transformers, BERT, and GPT models. Projects include building chatbots, text classifiers, and language generation systems.

## Prerequisites

- **CS021**: Prerequisite Course 21 (Minimum grade: C)

## Learning Objectives

- Understand core concepts in natural language processing
- Implement natural language processing algorithms and techniques
- Apply natural language processing to real-world problems
- Analyze and evaluate natural language processing solutions
- Design and build complete computer science systems
