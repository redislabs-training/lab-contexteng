# CS017: Mobile App Development

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: in_person
- **Instructor**: Holly Wood
- **Semester**: fall 2025
- **Max Enrollment**: 58

## Description

iOS and Android development with React Native. This course covers mobile UI design, navigation, state management, and API integration. Students will build and deploy cross-platform mobile applications.

## Learning Objectives

- Understand core concepts in mobile app development
- Implement mobile app development algorithms and techniques
- Apply mobile app development to real-world problems
- Analyze and evaluate mobile app development solutions
- Design and build complete computer science systems
