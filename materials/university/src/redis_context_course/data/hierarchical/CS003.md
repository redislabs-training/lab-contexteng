# CS003: Programming Fundamentals with C++

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: beginner
- **Format**: online
- **Instructor**: Angie Henderson
- **Semester**: summer 2025
- **Max Enrollment**: 68

## Description

Core programming concepts using C++ for beginners. This course introduces students to programming fundamentals using C++, focusing on computational thinking, problem-solving, and software development. No prior experience required. Students will learn variables, control structures, functions, arrays, pointers, object-oriented programming, and memory management through practical exercises.

## Learning Objectives

- Understand core concepts in programming fundamentals with c++
- Implement programming fundamentals with c++ algorithms and techniques
- Apply programming fundamentals with c++ to real-world problems
- Analyze and evaluate programming fundamentals with c++ solutions
- Design and build complete computer science systems
