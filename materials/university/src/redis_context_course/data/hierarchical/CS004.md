# CS004: Data Structures and Algorithms

## Course Information

- **Department**: Computer Science
- **Credits**: 4
- **Difficulty Level**: intermediate
- **Format**: hybrid
- **Instructor**: Daniel Wagner
- **Semester**: fall 2025
- **Max Enrollment**: 32

## Description

Essential data structures and algorithmic problem-solving. This course covers arrays, linked lists, stacks, queues, trees, graphs, and hash tables. Students will analyze algorithm complexity and implement efficient solutions to computational problems.

## Prerequisites

- **CS001**: Prerequisite for Data Structures and Algorithms (Minimum grade: C)

## Learning Objectives

- Understand core concepts in data structures and algorithms
- Implement data structures and algorithms algorithms and techniques
- Apply data structures and algorithms to real-world problems
- Analyze and evaluate data structures and algorithms solutions
- Design and build complete computer science systems
