# CS013: Deep Learning and Neural Networks

## Course Information

- **Department**: Computer Science
- **Credits**: 4
- **Difficulty Level**: advanced
- **Format**: online
- **Instructor**: Lisa Hensley
- **Semester**: winter 2025
- **Max Enrollment**: 63

## Description

Advanced neural network architectures and deep learning techniques. This course explores convolutional neural networks, recurrent networks, transformers, and generative models. Students will implement state-of-the-art architectures using PyTorch and apply them to real-world problems in vision, language, and beyond.

## Prerequisites

- **CS010**: Prerequisite for Deep Learning and Neural Networks (Minimum grade: C)

## Learning Objectives

- Understand core concepts in deep learning and neural networks
- Implement deep learning and neural networks algorithms and techniques
- Apply deep learning and neural networks to real-world problems
- Analyze and evaluate deep learning and neural networks solutions
- Design and build complete computer science systems
