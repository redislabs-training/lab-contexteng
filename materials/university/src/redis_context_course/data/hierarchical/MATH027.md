# MATH027: Differential Equations

## Course Information

- **Department**: Mathematics
- **Credits**: 3
- **Difficulty Level**: advanced
- **Format**: in_person
- **Instructor**: Justin Baker
- **Semester**: summer 2025
- **Max Enrollment**: 79

## Description

Ordinary differential equations and applications. This course covers first and second-order ODEs, systems of equations, and Laplace transforms. Applications include population dynamics, circuits, and mechanical systems.

## Prerequisites

- **MATH022**: Prerequisite for Differential Equations (Minimum grade: C)

## Learning Objectives

- Understand core concepts in differential equations
- Implement differential equations algorithms and techniques
- Apply differential equations to real-world problems
- Analyze and evaluate differential equations solutions
- Design and build complete mathematics systems
