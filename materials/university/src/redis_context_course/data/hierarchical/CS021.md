# CS021: Human-Computer Interaction

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: hybrid
- **Instructor**: Margaret Hawkins DDS
- **Semester**: summer 2025
- **Max Enrollment**: 46

## Description

User interface design, usability, and user experience. This course covers design principles, prototyping, user research, and evaluation methods. Students will design and test interactive systems.

## Learning Objectives

- Understand core concepts in human-computer interaction
- Implement human-computer interaction algorithms and techniques
- Apply human-computer interaction to real-world problems
- Analyze and evaluate human-computer interaction solutions
- Design and build complete computer science systems
