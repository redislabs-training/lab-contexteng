# CS010: Cybersecurity Fundamentals

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: in_person
- **Instructor**: Ryan Munoz
- **Semester**: summer 2025
- **Max Enrollment**: 62

## Description

Security principles, cryptography, and threat mitigation. This course introduces cybersecurity concepts including encryption, authentication, network security, and secure coding practices. Students will learn to identify and mitigate common vulnerabilities.

## Learning Objectives

- Understand core concepts in cybersecurity fundamentals
- Implement cybersecurity fundamentals algorithms and techniques
- Apply cybersecurity fundamentals to real-world problems
- Analyze and evaluate cybersecurity fundamentals solutions
- Design and build complete computer science systems
