# ARCH049: Historic Preservation

## Course Information

- **Department**: Architecture
- **Credits**: 3
- **Difficulty Level**: advanced
- **Format**: hybrid
- **Instructor**: Joshua Blair
- **Semester**: winter 2025
- **Max Enrollment**: 51

## Description

Conservation of historic buildings and cultural heritage. This course covers documentation, assessment, and restoration of historic structures. Students will learn preservation standards and develop conservation plans.

## Prerequisites

- **ARCH041**: Prerequisite for Historic Preservation (Minimum grade: C)

## Learning Objectives

- Understand core concepts in historic preservation
- Implement historic preservation algorithms and techniques
- Apply historic preservation to real-world problems
- Analyze and evaluate historic preservation solutions
- Design and build complete architecture systems
