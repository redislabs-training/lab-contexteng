# BIO038: Biochemistry

## Course Information

- **Department**: Biology
- **Credits**: 4
- **Difficulty Level**: advanced
- **Format**: hybrid
- **Instructor**: Cassandra Gaines
- **Semester**: winter 2025
- **Max Enrollment**: 58

## Description

Chemical processes and molecules of living organisms. This course covers the structure and function of biological macromolecules, enzyme kinetics, metabolism, and molecular signaling.

## Prerequisites

- **BIO030**: Prerequisite for Biochemistry (Minimum grade: C)

## Learning Objectives

- Understand core concepts in biochemistry
- Implement biochemistry algorithms and techniques
- Apply biochemistry to real-world problems
- Analyze and evaluate biochemistry solutions
- Design and build complete biology systems
