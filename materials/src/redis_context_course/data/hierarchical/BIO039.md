# BIO039: Neuroscience

## Course Information

- **Department**: Biology
- **Credits**: 4
- **Difficulty Level**: advanced
- **Format**: online
- **Instructor**: Elizabeth Fowler
- **Semester**: summer 2025
- **Max Enrollment**: 64

## Description

Brain structure, neural circuits, and behavior. This course explores the nervous system from molecules to behavior, covering neuroanatomy, synaptic transmission, sensory systems, and cognitive neuroscience.

## Prerequisites

- **BIO033**: Prerequisite for Neuroscience (Minimum grade: C)

## Learning Objectives

- Understand core concepts in neuroscience
- Implement neuroscience algorithms and techniques
- Apply neuroscience to real-world problems
- Analyze and evaluate neuroscience solutions
- Design and build complete biology systems
