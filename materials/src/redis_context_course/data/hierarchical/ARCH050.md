# ARCH050: Landscape Architecture

## Course Information

- **Department**: Architecture
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: in_person
- **Instructor**: Eric Carney
- **Semester**: spring 2025
- **Max Enrollment**: 35

## Description

Design of outdoor spaces, parks, and landscapes. This course covers site planning, planting design, grading, and landscape construction. Students will design parks, gardens, and public landscapes.

## Prerequisites

- **ARCH041**: Prerequisite for Landscape Architecture (Minimum grade: C)

## Learning Objectives

- Understand core concepts in landscape architecture
- Implement landscape architecture algorithms and techniques
- Apply landscape architecture to real-world problems
- Analyze and evaluate landscape architecture solutions
- Design and build complete architecture systems
