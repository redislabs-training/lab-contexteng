# ARCH051: Interior Architecture

## Course Information

- **Department**: Architecture
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: online
- **Instructor**: Jessica Holmes
- **Semester**: spring 2025
- **Max Enrollment**: 55

## Description

Interior space planning and design. This course focuses on the design of interior environments including residential, commercial, and institutional spaces. Topics include space planning, materials, lighting, and furniture.

## Prerequisites

- **ARCH041**: Prerequisite for Interior Architecture (Minimum grade: C)

## Learning Objectives

- Understand core concepts in interior architecture
- Implement interior architecture algorithms and techniques
- Apply interior architecture to real-world problems
- Analyze and evaluate interior architecture solutions
- Design and build complete architecture systems
