# ARCH045: Architectural Design Studio I

## Course Information

- **Department**: Architecture
- **Credits**: 4
- **Difficulty Level**: intermediate
- **Format**: online
- **Instructor**: Angelica Tucker
- **Semester**: summer 2025
- **Max Enrollment**: 40

## Description

Residential design and small-scale projects. This studio course focuses on designing single-family homes and small buildings. Students will develop design concepts, create drawings, and build physical models.

## Prerequisites

- **ARCH041**: Prerequisite for Architectural Design Studio I (Minimum grade: C)

## Learning Objectives

- Understand core concepts in architectural design studio i
- Implement architectural design studio i algorithms and techniques
- Apply architectural design studio i to real-world problems
- Analyze and evaluate architectural design studio i solutions
- Design and build complete architecture systems
