# CS002: Web Development Fundamentals

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: beginner
- **Format**: in_person
- **Instructor**: Noah Rhodes
- **Semester**: winter 2025
- **Max Enrollment**: 67

## Description

Introduction to web programming with HTML, CSS, and JavaScript. This beginner-friendly course teaches students how to build interactive websites from scratch. No prior programming experience required. Students will learn HTML structure, CSS styling, JavaScript programming, responsive design, and modern web development practices through hands-on projects.

## Learning Objectives

- Understand core concepts in web development fundamentals
- Implement web development fundamentals algorithms and techniques
- Apply web development fundamentals to real-world problems
- Analyze and evaluate web development fundamentals solutions
- Design and build complete computer science systems
