# ARCH046: Structures and Building Systems

## Course Information

- **Department**: Architecture
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: online
- **Instructor**: Philip Cannon
- **Semester**: spring 2025
- **Max Enrollment**: 34

## Description

Structural principles and mechanical systems. This course covers structural analysis, load paths, and building systems including HVAC, plumbing, and electrical. Students will design structural systems for buildings.

## Prerequisites

- **ARCH041**: Prerequisite for Structures and Building Systems (Minimum grade: C)

## Learning Objectives

- Understand core concepts in structures and building systems
- Implement structures and building systems algorithms and techniques
- Apply structures and building systems to real-world problems
- Analyze and evaluate structures and building systems solutions
- Design and build complete architecture systems
