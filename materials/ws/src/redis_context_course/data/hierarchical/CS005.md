# CS005: Database Systems

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: in_person
- **Instructor**: Cristian Santos
- **Semester**: summer 2025
- **Max Enrollment**: 74

## Description

Relational databases, SQL, and data modeling. This course covers database design, normalization, SQL programming, and transaction management. Students will design and implement databases for real-world applications.

## Learning Objectives

- Understand core concepts in database systems
- Implement database systems algorithms and techniques
- Apply database systems to real-world problems
- Analyze and evaluate database systems solutions
- Design and build complete computer science systems
