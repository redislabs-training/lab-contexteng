# BIO042: Bioinformatics

## Course Information

- **Department**: Biology
- **Credits**: 3
- **Difficulty Level**: advanced
- **Format**: hybrid
- **Instructor**: Fred Smith
- **Semester**: fall 2025
- **Max Enrollment**: 55

## Description

Computational analysis of biological data. This course teaches algorithms and tools for analyzing genomic, proteomic, and other biological datasets. Students will learn sequence alignment, phylogenetic analysis, and structural bioinformatics.

## Prerequisites

- **BIO030**: Prerequisite for Bioinformatics (Minimum grade: C)
- **CS001**: Prerequisite for Bioinformatics (Minimum grade: C)

## Learning Objectives

- Understand core concepts in bioinformatics
- Implement bioinformatics algorithms and techniques
- Apply bioinformatics to real-world problems
- Analyze and evaluate bioinformatics solutions
- Design and build complete biology systems
