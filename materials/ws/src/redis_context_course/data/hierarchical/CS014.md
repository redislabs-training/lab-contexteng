# CS014: Natural Language Processing

## Course Information

- **Department**: Computer Science
- **Credits**: 4
- **Difficulty Level**: advanced
- **Format**: hybrid
- **Instructor**: Michele Williams
- **Semester**: fall 2025
- **Max Enrollment**: 44

## Description

Text processing, language models, and NLP applications. This course covers the fundamentals of computational linguistics, from tokenization to large language models. Students will build text classifiers, named entity recognizers, and chatbots using modern NLP techniques and transformer-based models.

## Prerequisites

- **CS010**: Prerequisite for Natural Language Processing (Minimum grade: C)

## Learning Objectives

- Understand core concepts in natural language processing
- Implement natural language processing algorithms and techniques
- Apply natural language processing to real-world problems
- Analyze and evaluate natural language processing solutions
- Design and build complete computer science systems
