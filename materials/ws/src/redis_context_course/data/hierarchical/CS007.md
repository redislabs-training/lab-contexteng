# CS007: Software Engineering

## Course Information

- **Department**: Computer Science
- **Credits**: 3
- **Difficulty Level**: intermediate
- **Format**: online
- **Instructor**: Abigail Shaffer
- **Semester**: winter 2025
- **Max Enrollment**: 71

## Description

Software development lifecycle, design patterns, and best practices. This course covers requirements analysis, system design, testing, and maintenance. Students will work in teams to build a substantial software project using agile methodologies.

## Learning Objectives

- Understand core concepts in software engineering
- Implement software engineering algorithms and techniques
- Apply software engineering to real-world problems
- Analyze and evaluate software engineering solutions
- Design and build complete computer science systems
