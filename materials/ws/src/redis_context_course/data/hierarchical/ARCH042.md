# ARCH042: Architectural Drawing and Representation

## Course Information

- **Department**: Architecture
- **Credits**: 3
- **Difficulty Level**: beginner
- **Format**: hybrid
- **Instructor**: Fred Smith
- **Semester**: fall 2025
- **Max Enrollment**: 35

## Description

Technical drawing, CAD, and visualization techniques. This course teaches orthographic projection, perspective drawing, and computer-aided design using AutoCAD and Revit. Students will create professional architectural drawings.

## Learning Objectives

- Understand core concepts in architectural drawing and representation
- Implement architectural drawing and representation algorithms and techniques
- Apply architectural drawing and representation to real-world problems
- Analyze and evaluate architectural drawing and representation solutions
- Design and build complete architecture systems
