# BIO034: Introduction to Biology

## Course Information

- **Department**: Biology
- **Credits**: 3
- **Difficulty Level**: beginner
- **Format**: online
- **Instructor**: Nathan Maldonado
- **Semester**: winter 2025
- **Max Enrollment**: 73

## Description

Fundamental concepts of life, cells, and evolution. This introductory course covers cell structure, metabolism, genetics, evolution, and ecology. Designed for non-majors and beginning biology students.

## Learning Objectives

- Understand core concepts in introduction to biology
- Implement introduction to biology algorithms and techniques
- Apply introduction to biology to real-world problems
- Analyze and evaluate introduction to biology solutions
- Design and build complete biology systems
